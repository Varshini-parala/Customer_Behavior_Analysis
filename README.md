Overview:
This project analyzes customer purchasing data to identify meaningful patterns and business insights. It demonstrates an end-to-end data analytics workflow, combining SQL for data extraction, Python for analysis, and Power BI for visualization.
The goal of this project is to showcase practical data analyst skills that are commonly used in real-world business environments.

Key Objectives:
Analyze customer purchasing behavior
Identify revenue-driving customer segments
Evaluate product and category performance
Build an interactive dashboard for business users

Tools & Technologies:
SQL: Data extraction, joins, aggregations, metric validation
Python: Pandas, NumPy for analysis; Matplotlib, Seaborn for visualization
Power BI: Data modeling, KPIs, slicers, interactive dashboards
Jupyter Notebook: Exploratory Data Analysis (EDA)

Data Workflow:
Customer data stored in a SQL database
SQL queries used to extract and aggregate data
Python connected to SQL for EDA and validation
SQL database connected directly to Power BI
Dashboard created to present business insights
